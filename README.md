# Task Manager

A simple task management application using HTML, CSS, JavaScript, Express.js, and MySQL.

## Features
- Add, view, update, and delete tasks
- Backend: Node.js with Express.js
- Frontend: HTML, CSS, JavaScript
- Database: MySQL

4. Run the backend server:
